สรุปสูตรคณิตศาสตร์ ม.ปลาย

เรื่อง จำนวนเชิงซ้อน (Complex Numbers)

โดย อาจารย์ เอกชัย หลายศิริกุล ● ภาควิชาคณิตศาสตร์ คณะวิทยาศาสตร์ มหาวิทยาลัยนเรศวร

1 นิยามและหน่วยจินตภาพ (Definition & Imaginary Unit)

รูป Cartesian (พิกัดฉาก)

$$z = a + bi$$
  • $a = \text{Re}(z)$ คือ ส่วนจริง (Real Part)
  • $b = \text{Im}(z)$ คือ ส่วนจินตภาพ (Imaginary Part)
  • $i = \sqrt{-1}$ หรือ $i^2 = -1$

วัฏจักรของ $i^n$ (Powers of i)

วนลูปทุกๆ 4 ตัว (นำเลขชี้กำลัง $n$ หารด้วย 4 ดูเศษ)

เศษเหลือ ค่าที่ได้
เศษ 1$i$
เศษ 2$-1$
เศษ 3$-i$
หารลงตัว (เศษ 0)$1$

2 รูปแบบและการแปลง (Complex Forms)

1. รูปพิกัดฉาก (Rectangular)

$$z = a + bi$$

คู่ลำดับ $(a, b)$

2. รูปเชิงขั้ว (Polar Form)

$$z = r(\cos \theta + i \sin \theta)$$

เขียนย่อ: $z = r \text{cis } \theta$

การแปลงระหว่างรูปแบบ

หาขนาด ($r$ หรือ Modulus) $$r = |z| = \sqrt{a^2 + b^2}$$
หามุม ($\theta$ หรือ Argument) $$\tan \theta = \frac{b}{a}$$ (ดูจตุภาค Quadrant ประกอบเสมอ)

3 พีชคณิตและสมบัติ (Operations & Properties)

การคำนวณพื้นฐาน

กำหนด $z_1 = a+bi, z_2 = c+di$

  • บวก/ลบ: $$(a\pm c) + (b\pm d)i$$
  • คูณ: $$(ac - bd) + (ad + bc)i$$
  • หาร (คูณสังยุค): $$\frac{z_1}{z_2} = \frac{z_1 \cdot \bar{z}_2}{|z_2|^2}$$

สังยุค ($\bar{z}$) และ ค่าสัมบูรณ์ ($|z|$)

Conjugate
$\overline{a+bi} = a-bi$
Modulus
$|z| = \sqrt{a^2+b^2}$
สมบัติสำคัญ:
  • $|z|^2 = z \cdot \bar{z}$ (สำคัญ!)
  • $\overline{z_1 \pm z_2} = \bar{z}_1 \pm \bar{z}_2$
  • $|z_1 z_2| = |z_1| |z_2|$
  • $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$

4 เชิงขั้ว, เดอมัวร์ และราก (Polar, De Moivre & Roots)

การดำเนินการรูปเชิงขั้ว

กำหนด $z_1 = r_1 \text{cis } \theta_1$ และ $z_2 = r_2 \text{cis } \theta_2$

การคูณ $r_1 r_2 \text{cis } (\theta_1 + \theta_2)$
การหาร $\frac{r_1}{r_2} \text{cis } (\theta_1 - \theta_2)$
ยกกำลัง (De Moivre) $z^n = r^n \text{cis } (n\theta)$

การหารากที่ $n$ ($w^n = z$)

$$w_k = \sqrt[n]{r} \left[ \cos \left( \frac{\theta + 2k\pi}{n} \right) + i \sin \left( \frac{\theta + 2k\pi}{n} \right) \right]$$

โดยที่ $k = 0, 1, 2, ..., n-1$

ข้อสังเกต: รากทั้ง $n$ คำตอบ จะมีขนาดเท่ากัน (อยู่บนวงกลมรัศมี $\sqrt[n]{r}$) และมุมห่างกันเท่าๆ กันเป็นมุม $\frac{360^\circ}{n}$

5 สมการพหุนาม (Polynomial Equations)

สมการกำลังสอง $ax^2 + bx + c = 0$

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

ถ้า $b^2 - 4ac < 0$ คำตอบจะเป็นจำนวนเชิงซ้อนที่เป็นคู่สังยุคกัน (Conjugate Pairs)
